Root Level:(Static php pages that are used to generate html pages)
  • index.phps is the main page. Its content is transfered to browse.phps
  • album.phps displays the album info.
  • register.phps is responsible for registering and logging in.
  • Includes Folder:(Php codes that connects to db)
    • config.phps is responsible for connecting to the mysql server.
    • footer.phps is responsible for creating the player bar and other section for each page.
    • header.phps is responsible for creating the left portion(Logo, Browse) for each page.
    • includedFiles.phps is responsible for checking if the user typed the url or the Jquery triggered it.
    • navBarContainer.phps is responsible for display only the Logo section and browse button for every page.
    • nowPlayingBar.phps is responsible for display the player section for each page.
    • Classes Folder:
      • Account.phps, Album.phps, Artist.phps and Song.phps are used to obtain relevent data from database.
      • Constant.phps is storing error messages for incorrent format in register page.
    • Handlers Folder:
      • login-handler.phps forward user to register page if they are not logged in.
      • register-handler.phps cleans user input and thus increase security.
      • getAlbumjson.phps, getArtistjson.phps and getSongjson.phps gets the data from database into the playlist via ajax. updatePlays.phps updates the song played times in database via ajax.
